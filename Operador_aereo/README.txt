El programa anailiza una pequeña BigData
El código está escrito para python 3.7
Tarda entre 1 min y 2 la primera vez que se instala, después es todo instantáneo
La única libreria externa, la cual se puede instalar con 'pip', es matplotlib y pygame
Este programa solo funciona descomprimiendo la carpeta data
La información se da en formato:
	Nombre de archivo: 201809DD_VVVV_AAA.txt donde DD es el día, VVVV el vuelo, el cual puede ser o 2892 o 3745 u el AAA es el asiento que puede tener va de 1 a 40 y la letra del asiento puede ser A,B,C o D (Mire los que están  adjuntos en data para aclarar)
	El .txt tiene que tener dentro:
		DNI
		Dinero gastado
		Fecha de nacimiento en formato YYYYMMDD
		Sexo M/F
